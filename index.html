GPXSee Flatpak

Unofficial Flatpak bundles for GPXSee, the GPS log file viewer and analyzer.

What is this?

This page distributes standalone Flatpak bundles of GPXSee, built from the manifest in this repository (joklaps/GPXSee-flatpak). These builds are not affiliated with or endorsed by the upstream GPXSee project, and are not published on Flathub — they are single-file .flatpak bundles you install directly.

Requirements

  1. Flatpak installed on your system — see the official setup instructions for your distribution.
  2. The Flathub remote added, since the bundle depends on the KDE runtime hosted there: 
    flatpak remote-add --user --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo

Installing a downloaded bundle

Download the .flatpak file for your architecture from the table below, then run:

flatpak install --user ./gpxsee_x86_64_v16.11.flatpak

Once installed, launch it with:

flatpak run org.gpxsee.GPXSee

Updating: these bundles are plain files, not served from a Flatpak remote, so flatpak update won't pick up new versions. To upgrade, download the newer bundle and install it again with the same command above — it will replace the existing installation.
